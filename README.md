# My Claude Skills

Personal collection of Claude Code skills — backed up from all machines.

## Install one skill

```bash
npx skills add spiosifidis/my-claude-skills -s skill-name
```

## Install all skills

```bash
npx skills add spiosifidis/my-claude-skills -s '*'
```
