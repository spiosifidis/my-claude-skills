# Food ordering, substitution notes, and naming conventions

## 1. Food order in a meal (by dish type)

Order mirrors the dish's main ingredient first, then supporting ingredients, then salad/dressing, then oil last. When the client's own questionnaire wording gives an explicit order for a specific dish, follow that instead of the table below.

| Dish type | Order |
|---|---|
| Λαδερά (φασολάκια, μπριάμ κ.λπ.) | Κύριο λαχανικό → Ψωμί → Φέτα |
| Κοτόπουλο/κρέας με ρύζι/πατάτα | Κύριο κρέας → Ρύζι/Πατάτα → Σαλάτα → Ελαιόλαδο |
| Μακαρόνια με κιμά | Μακαρόνια → Κιμάς → Σαλάτα → Ελαιόλαδο |
| Μακαρόνια με κιμά + γραβιέρα | Μακαρόνια → Κιμάς → Γραβιέρα → Σαλάτα → Ελαιόλαδο |
| Καλαμάκι / Wrap / Πίτα | Πίτα/Wrap ΠΡΩΤΑ → Κρέας/Κοτόπουλο → υπόλοιπα |
| Αυγά + τυρί + ψωμί | Αυγό → Τυρί → Παξιμάδι/Ψωμί → Σαλάτα → Ελαιόλαδο |
| Ντάκο | Παξιμάδι → Φέτα → Ελαιόλαδο → Ντομάτα |

**Κανόνας**: όπου υπάρχει σαλάτα επιλογής, υπάρχει και ελαιόλαδο (1-2 κ.γ.).

### Γεύμα-σαλάτα (marked "(σαλάτα)")

Όταν η διαιτολόγος γράφει ένα γεύμα-σαλάτα (πχ "Σαλάτα επιλογής με κοτόπουλο ή βραστά αυγά, παξιμάδι ολικής, καλαμπόκι βραστό, κρέμα βαλσάμικου και ελαιόλαδο"), το σημαδεύει με **"(σαλάτα)"** σε παρένθεση στο τέλος των σημειώσεων του πελάτη. Σειρά:

1. **Σαλάτα επιλογής** πρώτη
2. **Πρωτεΐνη** (αυγά, γαρίδες, τόνος, κοτόπουλο)
3. Ό,τι άλλο υπάρχει (παξιμάδι/φρυγανιά, καλαμπόκι κ.λπ.)
4. **Κρέμα βαλσάμικου** (αν υπάρχει στο γεύμα) — προς το τέλος, πριν το λάδι
5. **Ελαιόλαδο** τελευταίο

Ό,τι δεν υπάρχει στο συγκεκριμένο γεύμα παραλείπεται — προχωράς στο επόμενο τρόφιμο της σειράς, δεν αφήνεις κενό.

### Μακαρονοσαλάτα

Όταν το γεύμα γράφει "μακαρονοσαλάτα με …": **Μακαρόνια βρασμένα** πρώτα → **πρωτεΐνη** (αυγά, γαρίδες, τόνος, κοτόπουλο) → ό,τι άλλο ζητηθεί → **μαγιονέζα** (όποιο είδος ζητηθεί — κανονική ή light) → **ελαιόλαδο** αν ζητηθεί, τελευταίο. Ό,τι λείπει παραλείπεται, δεν αφήνεις κενό στη σειρά.

### Πατατοσαλάτα

Όταν το γεύμα γράφει "πατατοσαλάτα με …": **Πατάτα βραστή** πρώτη → **πρωτεΐνη** (αυγά, γαρίδες, τόνος, κοτόπουλο) → ό,τι άλλο ζητηθεί → **μαγιονέζα** (όποιο είδος ζητηθεί) ή/και **ελαιόλαδο** αν ζητηθεί, τελευταία. Ό,τι λείπει παραλείπεται.

### Σαλάτα με κινόα

Όταν το γεύμα γράφει "σαλάτα με κινόα …": **Κινόα βρασμένη** πρώτη → **πρωτεΐνη** (αυγά, γαρίδες, τόνος, κοτόπουλο) → ό,τι άλλο ζητηθεί → **μαγιονέζα** (όποιο είδος ζητηθεί) ή/και **ελαιόλαδο** αν ζητηθεί, τελευταία. Ό,τι λείπει παραλείπεται.

## 2. Substitution notes

- Γράφονται στις **σημειώσεις κάτω από το γεύμα** (plain text, χωρίς HTML).
- "Τρόφιμο Α ή Τρόφιμο Β" → η σημείωση γράφει την **αντίστοιχη ποσότητα του Β**, ισοδύναμη σε θερμίδες με το Α που μπήκε στο γεύμα.
- **Εξαίρεση — ρητή ποσότητα**: αν ο πελάτης έχει ήδη δώσει συγκεκριμένη ποσότητα για την εναλλακτική (πχ "ή 2 αυγά"), χρησιμοποιείται αυτή, όχι υπολογισμένη ισοδυναμία.
- **Εξαίρεση — "ίδια γραμμάρια"**: αν ο πελάτης ζητά ίδια γραμμάρια (όχι ίδιες θερμίδες), γράφεται απλά η ίδια ποσότητα.
- **Πολλαπλές σημειώσεις υποκατάστασης** στο ίδιο γεύμα διαχωρίζονται με **"/"** (όχι newline, όχι "και").
- Κάπαρη, μουστάρδα, κανέλα κ.λπ. → στις σημειώσεις (πχ. `+κανέλα`).
- **Σημαντικό**: αντικαταστάσεις μπαίνουν στις σημειώσεις **μόνο** αν τις έχει γράψει η διαιτολόγος στο ερωτηματολόγιο. Δεν προστίθενται εναλλακτικές αυτόβουλα.
- **Τρόφιμο άγνωστο στη λίστα 4**: αν το γεύμα που ζητάει η διαιτολόγος περιλαμβάνει τρόφιμο που δεν υπάρχει στο [food-database.md](food-database.md) ούτε σε live `search_foods` **με `filters: { list: 4 }`**, **σταμάτα** και ζήτα από τη διαιτολόγο να το προσθέσει στη λίστα 4 στο Athlisis πρώτα — μην το αντικαθιστάς σιωπηλά με κάτι παρόμοιο και μην το ψάξεις/χρησιμοποιήσεις από άλλη λίστα (0, 1, 2, 5, συνταγές, αγαπημένα).

## 3. Naming conventions (client's own shorthand)

- **"Φρυγανιά" / "Φρυγανιές"** (χωρίς άλλο προσδιορισμό) → σημαίνει πάντα τις **φρυγανιές κλασσικές / Παπαδοπούλου Χωριάτικες** (ID `79605`, 48 kcal/τεμ).
- **"Wasa"** (ρητά ονομασμένο) → **Wasa φρυγανιές ολικής άλεσης** (ID `2997`, 40.1 kcal/τεμ).
- Μην τα μπερδεύεις — παλιότερα υπήρχε λανθασμένη σύμβαση που έβαζε Wasa ως default για "φρυγανιά" χωρίς προσδιορισμό. Διορθώθηκε 2026-07 (οδηγία διαιτολόγου).

## 4. Food grouping

- **Λαδερά λαχανικά** (Φασολάκια/Μπριάμ/Μπάμιες): το ένα είναι το κύριο τρόφιμο στο γεύμα· τα άλλα μπαίνουν στη σημείωση ως εναλλακτικά (ισοδύναμα σε θερμίδες, εκτός αν ζητηθούν "ίδια γραμμάρια" — τότε ίδια ποσότητα, βλ. §2).
- **Όσπρια**:
  - **Φακές** = κύρια επιλογή (με φέτα). Στις σημειώσεις: `ή Χγρ ρεβίθια ή γίγαντες ή φασολάδα`.
  - **Αρακάς** και **Σπανακόρυζο** = ξεχωριστές επιλογές· η κάθε μία αναφέρει την άλλη ως αντικατάσταση στις σημειώσεις.
- **Μακαρόνια & Κριθαράκι**: αν οι θερμίδες είναι πολύ κοντά, μπαίνουν μαζί με σημείωση εναλλαγής. Αν διαφέρουν σημαντικά, τυπική αντικατάσταση στις σημειώσεις (equivalence table στο [portion-ranges-and-equivalents.md](portion-ranges-and-equivalents.md)).
- **Φρούτο επιλογής**: ~60 kcal, 1 μέτριο τεμάχιο. ID `149492` (list 4) — αυτό και μόνο αυτό. (Το παλιό fallback σε ID `15022` list 5 "Φρούτο εποχής" δεν ισχύει πλέον — δες πολιτική "μόνο λίστα 4" στο [food-database.md](food-database.md).)

## 5. Meal-choice ordering within Μεσημεριανό & Βραδινό

Όταν χτίζεις τις επιλογές (choices/days) μέσα σε μία στήλη Μεσημεριανού ή Βραδινού, η ίδια σειρά προτεραιότητας ισχύει και για τα δύο:

1. Φασολάκια / μπριάμ / μπάμιες
2. Κοτόπουλο με συνοδευτικό
3. Όσπρια (φακές, ρεβίθια, φασολάδα κ.λπ.)
4. Αρακάς / Σπανακόρυζο
5. Μπιφτέκια με συνοδευτικό
6. Ψάρι με συνοδευτικό
7. Μακαρόνια με κιμά / κόκκινη σάλτσα
8. Μοσχάρι / αρνί / μπριζόλα με συνοδευτικό
9. Παστίτσιο / μουσακάς / γεμιστά / γιουβαρλάκια

**Κανόνας ίδιου φαγητού σε μεσημεριανό + βραδινό**: αν το ίδιο φαγητό εμφανίζεται και στις δύο λίστες του πελάτη, μπαίνει στην **ίδια choice/στήλη** (ίδιο `day` index) και για τα δύο meal_type. Αν λείπει από τη μία λίστα, παραλείπεται εκείνη η θέση και προχωράμε στην επόμενη — δεν μετακινούνται τα υπόλοιπα. (Παράδειγμα: πλάνο Παπακωνσταντίνου — "σαλάτα με κοτόπουλο + παξιμάδι + ελαιόλαδο" εμφανίστηκε και στα δύο, μπήκε στο ίδιο `day` index και στο Μεσημεριανό (1) και στο Βραδινό (1).)
