# Food database — "Τρόφιμά μου" (list 4)

Full catalog of the dietitian's own food list in Athlisis (~370+ foods, IDs, kcal, serving unit). Check here **before** calling `search_foods` — only search live for foods not listed below, or when an ID here errors with `invalid_food_id` (the app can rename/remove foods; patch this file when that happens).

**Only list 4** ("Τρόφιμά μου") is used — never list 0, 1, 2, 5, recipes, aCloud foods, or favorites. If a food isn't in this file, do a live `search_foods` filtered to `list: 4` only. If it still doesn't turn up, **stop and ask the dietitian to add it to list 4 in Athlisis first** — don't substitute silently and don't pull it from another list (see [food-order-and-substitutions.md](food-order-and-substitutions.md) §2). This is a hard rule as of 2026-07 — earlier versions of this file described a 5-list fallback chain; that's no longer used.
